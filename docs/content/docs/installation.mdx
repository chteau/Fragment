<Steps>
<Step>

### Add the ModuleScript

In **ReplicatedStorage**, insert a new `ModuleScript` and name it `Fragment`.

</Step>
<Step>

### Paste the source

Open the ModuleScript and paste the [Fragment source code](https://github.com/chteau/Fragment/blob/main/Fragment.luau) into it.

</Step>
<Step>

### Create your project folder

Create a folder somewhere under **StarterPlayerScripts** — for example `MySuperDuperUI`. This is where your Handles, Stores, Hooks and Contexts will live.

</Step>
</Steps>

## Initialize Fragment

Create `FragmentLoad.client.luau` as a `LocalScript` inside your project folder. This is the only entry point Fragment needs.

```luau title="FragmentLoad.client.luau"
local ReplicatedStorage = game:GetService("ReplicatedStorage")
local Fragment = require(ReplicatedStorage.Fragment)

Fragment.load(script.Parent.Handles) -- [!code highlight]
```

`Fragment.load` walks the folder recursively, requires every `ModuleScript` it finds, and ignores anything that does not return a Handle. After loading, it runs an initial render pass on all registered Handles.

```luau title="FragmentLoad.client.luau"
local ReplicatedStorage = game:GetService("ReplicatedStorage")
```
