# Properties

## `Fragment.DELETE`

Sentinel value used to delete context keys.

<TypeTable
  type={{
    DELETE: {
      description: "Sentinel value used to delete context keys.",
      type: "any"
    }
  }}
/>

**Example**:
```luau title="WindowContext.luau"
WindowContext._value.toggle = function(name: string)
    if WindowContext._value.active == name then
        WindowContext:patch({ active = Fragment.DELETE }) -- Sets active to "nil" [!code highlight]
    else
        WindowContext:patch({ active = name })
    end
end
```

## `createStore(name, builder)`

Creates a new reactive store.

<TypeTable
  type={{
    createStore: {
      description: "Creates a new reactive store.",
      type: "(name: string, builder: (set: (partial: any) -> (), get: () -> any) -> any) -> any"
    }
  }}
/>

**Example**:
```luau title="ReplicaServiceStore.luau"
return Fragment.createStore("ReplicaServiceStore", function(set, get)
    return {
        ReplicaData = {},

        setData = function(newData: table)
            set({ ReplicaData = newData })
        end,

        getData = function()
            return get().ReplicaData
        end,
    }
end)
```

## `newComponent(name, path)`

Creates a new reusable Component definition. The path resolves from the DataModel root (`game`), since Components are typically stored in assets folders.

<TypeTable
  type={{
    newComponent: {
      description: "Creates a new reusable Component definition bound to a template path resolved from game root.",
      type: "(name: string, path: { string }) -> Component"
    }
  }}
/>

Call the returned Component with a renderer function to complete registration. The renderer receives:
- **element** — the cloned template instance
- **props** — the props table passed to `useComponent`
- **handle** — the calling Handle, giving full access to `Effect`, `Connect`, `RegisterContainer`, stores, contexts, and hooks

**Example**:
```luau title="Components/SellItem.luau"
local Component = Fragment.newComponent("SellItem", {
    "ReplicatedStorage", "BE_Shared", "Assets", "UI", "SellTemplate"
})

return Component(function(element: Frame, props: {
    Id:    string,
    Price: number,
}, handle: any)
    element.Name = props.Id
    element:FindFirstChild("PriceLabel").Text = `🪙 {props.Price}`

    handle.Connect("Buy_" .. props.Id, element.BuyBtn.Activated, function()
        -- ...
    end)
end)
```

## `useComponent(name, props)`

Clones the template for a registered Component, runs its renderer with the given props and the active Handle, and returns the unparented clone.

<TypeTable
  type={{
    useComponent: {
      description: "Clones and renders a registered Component. Returns the unparented instance, or nil on failure.",
      type: "(name: string, props: { [string]: any }) -> Instance?"
    }
  }}
/>

**Example**:
```luau title="Handles/ShopHandle.luau"
local container = Handle.RegisterContainer(element.ItemGrid)

for _, entry in ipairs(data.items) do
    local item = Fragment.useComponent("SellItem", { -- [!code highlight]
        Id    = entry.Id, -- [!code highlight]
        Price = entry.Price, -- [!code highlight]
    }) -- [!code highlight]
    container.UpsertChild(entry.Id, item)
end
```

## `load(target)`

Loads all Fragment modules and fires the first render pass.

<TypeTable
  type={{
    load: {
      description: "Loads Fragment modules in dependency order and fires the initial render pass.",
      type: "(target: Instance | { Stores: Instance?, Contexts: Instance?, Hooks: Instance?, Components: Instance?, Handles: Instance? }) -> ()"
    }
  }}
/>

Accepts either a single folder (legacy) or a structured table of folders loaded in guaranteed dependency order: `Stores → Contexts → Hooks → Components → Handles`. All keys are optional.

**Structured usage**:
```luau title="FragmentLoad.client.luau"
Fragment.load({ -- [!code highlight]
    Stores     = script.Parent.Stores, -- [!code highlight]
    Contexts   = script.Parent.Contexts, -- [!code highlight]
    Hooks      = script.Parent.Hooks, -- [!code highlight]
    Components = script.Parent.Components, -- [!code highlight]
    Handles    = script.Parent.Handles, -- [!code highlight]
}) -- [!code highlight]
```

**Legacy usage** — retro-compatible:
```luau title="FragmentLoad.client.luau"
Fragment.load(script.Parent.Handles)
```
